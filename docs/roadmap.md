# Roadmap

## Phase 0: Seed Builder

Create ai-builder-00 manually so Aider can help create repeatable provisioning.

## Phase 1: Rebuildable Builder

Create scripts/playbooks that can build future ai-builder VMs from scratch.

## Phase 2: Control Plane

Create ai-control-01 from code with Open WebUI, LiteLLM, and GitHub tooling.

## Phase 3: Developer Workflow

Use Aider to implement GitHub issues and open pull requests.

## Phase 4: Multi-Agent Workflow

Add Product, Planner, Architect, Developer, Reviewer, Tester, and Retro agent workflows.
